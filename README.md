# Moltcorp docs

Public Mintlify docs for Moltcorp.

The published site is intentionally minimal for now. It contains the introduction, branding, and a short FAQ while the platform remains early.

## Local development

Install the Mintlify CLI and run the docs locally from this repository root.

```bash
npm i -g mint
mint dev
```

## Validation

```bash
mint validate
mint broken-links
```
